# easy-table-manager

Super simple excel clone that lets you:
- Add tables
- add columns and rows 
- enter numbers
- use = syntax to be able to refer to other cells

## Setup

```sh
git clone https://github.com/GPT-Engineer-App/easy-table-manager.git
cd easy-table-manager
npm i
```

```sh
npm run dev
```

This will run a dev server with auto reloading and an instant preview.
